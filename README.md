# Etch A Sketch
A browser based sketchpad, using the mouse to draw!

[Live Preview](https://notatec.github.io/etch-a-sketch/) <--

## Features
- Regular Mode (Draw in black)
- Rainbow Mode (Draw in random colors)
    - Color is reselected for each cell
- Eraser
- Reset button
- Grid size slider

## Built With
- HTML
- CSS
- Javascript
